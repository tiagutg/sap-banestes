# SAP - Sistema de Acompanhamento de Processos (Banestes)

O **SAP** é uma aplicação web de página única (SPA) desenvolvida para o gerenciamento eficiente de processos. O sistema utiliza o Google Sheets como banco de dados em tempo real e oferece uma interface moderna, responsiva e intuitiva para operações de CRUD (Criar, Ler, Atualizar e Excluir).

---

## 🚀 Funcionalidades Principais

- **Consulta Inteligente:** Localização instantânea de registros através do ID do processo.
- **Gerenciamento de Dados:** Interface completa para edição de valores, alteração de status e remoção de registros.
- **Interface Responsiva:** Design limpo desenvolvido com **Tailwind CSS**, adaptável a diferentes tamanhos de tela.
- **Feedback ao Usuário:** Sistema de notificações customizadas (Toasts) para confirmação de ações e alertas de erro.
- **Arquitetura Cloud:** Executado inteiramente na infraestrutura do Google Apps Script, eliminando a necessidade de servidores externos.

---

## 🛠️ Tecnologias Utilizadas

- **Frontend:** HTML5, JavaScript (ES6+), Tailwind CSS (via CDN).
- **Backend:** Google Apps Script (GAS).
- **Banco de Dados:** Google Sheets (Planilhas Google).
- **Ícones:** Font Awesome.

---

## 📦 Como Visualizar o Projeto

Para replicar este ambiente ou analisar o código:

1. **Acesse a Planilha:** Verifique a estrutura das colunas na planilha vinculada.
2. **Abra o Script:** O código lógico reside no arquivo `Código.gs` e a interface no `index.html`.
3. **Execução:** O sistema é acessado via URL de implantação do Apps Script (Web App).
